Cara Pakai Setelah Generate

  1. Generate gambar di leonardo.ai dengan prompt di atas.
  2. Download dalam format JPG atau WebP (quality 85%+).
  3. Rename sesuai nama file yang dicantumkan (contoh: about-founder.jpg).
  4. Simpan di folder yang sama dengan index.html: c:\Users\hebir\OneDrive\Documents\MONEYMAGNET\website\bazi-03\
  5. Di index.html, ganti URL Unsplash dengan nama file lokal sesuai komentar HTML yang sudah ditandai.
  6. Preview di browser untuk verifikasi tone dan ukuran.
Recommended settings di Leonardo: Model: Phoenix / Kino XL  |  Negative prompt: "text, watermark, logo, neon, rainbow, cartoon, illustration, 3D render"  |  Steps: 30+  |  Guidance scale: 7–9
